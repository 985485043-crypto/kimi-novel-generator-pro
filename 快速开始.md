# Kimi 15000字小说生成器 - 快速开始（5分钟）

## 最简单的启动方式

### Windows 用户

1. **安装 Node.js**
   - 下载：https://nodejs.org/ （选择 LTS 版本）
   - 安装时勾选 "Add to PATH"

2. **打开 PowerShell**
   - 在项目文件夹中按 `Shift + 右键` → 选择 "在此处打开 PowerShell"

3. **运行以下命令**
   ```bash
   npm install
   npm run dev
   ```

4. **打开浏览器**
   - 访问 `http://localhost:3000`

### macOS 用户

1. **安装 Node.js**
   ```bash
   brew install node
   ```

2. **进入项目文件夹**
   ```bash
   cd /path/to/kimi_novel_generator
   ```

3. **启动应用**
   ```bash
   npm install
   npm run dev
   ```

4. **打开浏览器**
   - 访问 `http://localhost:3000`

### Linux 用户

```bash
# 安装 Node.js
sudo apt-get install nodejs npm

# 进入项目文件夹
cd /path/to/kimi_novel_generator

# 启动应用
npm install
npm run dev
```

## 使用应用

1. 输入 Moonshot API Key（从 https://platform.moonshot.cn 获取）
2. 输入小说标题（例如："时空穿梭者"）
3. 选择创作类型（都市言情、虐恋、家庭、悬疑、甜宠）
4. 点击"开始生成小说"
5. 等待生成完成，点击"下载小说"保存

---

## 遇到问题？

**问题：命令不识别**
- 检查 Node.js 是否正确安装：`node --version`

**问题：端口被占用**
- 修改 `package.json` 中的端口号，或关闭其他占用 3000 端口的应用

**问题：API Key 错误**
- 确保 API Key 是有效的，访问 https://platform.moonshot.cn 检查
